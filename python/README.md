# Python Programs

This directory contains basic Python programs for learning and practice.

## Programs

### 1. hello_world.py
A simple "Hello, World!" program.

**Usage:**
```bash
python3 hello_world.py
```

### 2. calculator.py
A command-line calculator that performs basic arithmetic operations.

**Usage:**
```bash
python3 calculator.py
```

**Features:**
- Addition
- Subtraction
- Multiplication
- Division (with zero-division handling)

### 3. file_operations.py
Demonstrates file I/O operations including reading, writing, and appending.

**Usage:**
```bash
python3 file_operations.py
```

**Features:**
- Write to file
- Read from file
- Append to file
- File cleanup

### 4. sorting.py
Implementation of various sorting algorithms.

**Usage:**
```bash
python3 sorting.py
```

**Algorithms:**
- Bubble Sort
- Selection Sort
- Insertion Sort

## Requirements
- Python 3.6 or higher
