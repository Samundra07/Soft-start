# soft-start
Software engineering practicing, lecture and work.

## Overview
This repository contains a collection of programs in different programming languages for learning and practicing software development concepts.

### JavaScript Programs (`/javascript`)

See [javascript/README.md](javascript/README.md) for detailed information.

### C Programs (`/c`)
Collection of C programs demonstrating:
- Basic syntax
- Data structures
- Memory management

See [c/README.md](c/README.md) for detailed information.

## Getting Started

### Prerequisites
- Python 3.6+ for Python programs
- Node.js v12+ for JavaScript programs
- GCC compiler for C programs

### Running Programs

**Python:**
```bash
cd python
python3 <program_name>.py
```

**JavaScript:**
```bash
cd javascript
node <program_name>.js
```

**C:**
```bash
cd c
gcc <program_name>.c -o <output_name>
./<output_name>
```
