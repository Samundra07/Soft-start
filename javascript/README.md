# JavaScript Programs

This directory contains basic JavaScript/Node.js programs for learning and practice.

## Programs

### 1. hello_world.js
A simple "Hello, World!" program.

**Usage:**
```bash
node hello_world.js
```

### 2. simple_server.js
A basic HTTP server that handles different routes.

**Usage:**
```bash
node simple_server.js
```

Then open your browser and navigate to:
- http://localhost:3000/ - Home page
- http://localhost:3000/about - About page

Press Ctrl+C to stop the server.

### 3. array_operations.js
Demonstrates various array manipulation methods.

**Usage:**
```bash
node array_operations.js
```

**Features:**
- Map
- Filter
- Reduce
- Find
- Sort
- Reverse
- Slice
- Concat
- And more!

## Requirements
- Node.js (v12 or higher)
